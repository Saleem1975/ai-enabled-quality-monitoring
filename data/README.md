# Data Notes

This folder contains summary and demonstration datasets prepared from the published Phase I control chart pattern-recognition paper.

## Included files

### `model_comparison.csv`
Reported accuracy and standard deviation for ANN, 1D-CNN benchmark models, improved 1D-CNN, and the proposed m2D-CNN.

### `control_chart_patterns.csv`
The nine Phase I control chart patterns classified by the proposed model.

### `experiment_summary.csv`
Summary of the data-generation and experimental setup.

### `m2d_cnn_hyperparameters.csv`
Main hyperparameters reported for the proposed m2D-CNN model.

### `confusion_summary_notes.csv`
Key notes from the confusion matrix discussion.

### `sample_control_chart_patterns.csv`
Demonstration data showing example synthetic patterns for visualization only.

## Note

The included data files are summary/demo files based on values reported in the paper and are intended for educational, research, and portfolio demonstration purposes.
