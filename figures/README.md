# Figure Guide

Recommended README placement:

1. `project15_quality_monitoring_summary.jpg`
   - Main visual for the project and website card.
   - Shows model accuracy comparison and the headline quality-monitoring value.

2. `01_multimodal_cnn_architecture.jpg`
   - Place in the Methodology section.
   - Explains the m2D-CNN architecture and multimodal data fusion.

3. `02_model_accuracy_comparison.jpg`
   - Place in the Model Evaluation section.
   - Compares ANN, 1D-CNN benchmarks, improved 1D-CNN, and the proposed m2D-CNN.

4. `03_model_stability_comparison.jpg`
   - Shows standard deviation of model accuracy across repeated trials.

5. `04_simulated_control_chart_patterns.jpg`
   - Shows examples of the nine simulated Phase I control chart patterns.

6. `05_data_generation_evaluation_pipeline.jpg`
   - Shows the data simulation, image generation, statistic computation, and evaluation workflow.

7. `06_accuracy_vs_stability_tradeoff.jpg`
   - Shows the relationship between accuracy and variability across models.
