# Executive Summary

## Project title

**AI-Enabled Quality Monitoring: Phase I Control Chart Pattern Recognition Using Multimodal 2D-CNN**

## Summary

This project demonstrates how deep learning can support quality monitoring by automatically classifying Phase I control chart patterns. The model combines control chart images, histogram images, and numerical statistics into a unified multimodal learning framework.

## Business problem

SPC control charts are essential for monitoring process stability. However, manually identifying control chart patterns can be subjective and inconsistent, especially when patterns are complex or subtle.

## Proposed solution

The proposed m2D-CNN model integrates:

- Control chart visual information
- Histogram visual information
- Numerical process statistics

This fusion improves pattern recognition and supports faster quality diagnosis.

## Key result

The proposed m2D-CNN achieved 99.21% accuracy across nine Phase I control chart patterns, outperforming ANN and 1D-CNN benchmark models.

## Consulting relevance

The approach can support AI-enabled quality monitoring, early abnormality detection, defect prevention, and manufacturing process improvement.
